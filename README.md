### MFModeller
[![Ruff](https://img.shields.io/endpoint?url=https://raw.githubusercontent.com/astral-sh/ruff/main/assets/badge/v2.json)](https://github.com/astral-sh/ruff)
[![pyrefly](https://img.shields.io/endpoint?url=https://pyrefly.org/badge.json)](https://github.com/facebook/pyrefly)

This repository contains all code relating to my PhD thesis titled: ``Gaussian Process Based Multi-Fidelity
Modelling in Fluid Dynamics``. 

It also can serve as a starting point for any other researchers keen on continuing multi-fidelity research, especially for cases where there are some real data, and there is a desire to make this into a fully functioning test function for multi-fidelity modelling.

## Installation 
```bash
# Using uv (HTTPS)
uv add git+https://github.com/MarkPellowe/MFModeller.git

# Using uv (SSH)
uv add git+ssh://git@github.com//MarkPellowe/MFModeller.git

# Pin to a specific version
uv add git+ssh://git@github.com//MarkPellowe/MFModeller.git@
```
